# 🤖 Chatly – Real-Time AI Chatbot

A modern real-time AI chatbot built using **FastAPI**, **WebSockets**, **HTML**, **CSS**, and **JavaScript**. The application provides an interactive chat interface with instant responses and demonstrates real-time communication between the client and server.

---

## 🚀 Features

* 💬 Real-time messaging using WebSockets
* ⚡ FastAPI backend for high performance
* 🎨 Responsive and modern user interface
* 🌐 Jinja2 template rendering
* 🤖 Rule-based chatbot responses
* 👋 Handles greetings and small talk
* 😂 Built-in joke responses
* 🌦️ Weather-related response support (demo)
* 📱 Mobile-friendly interface
* 🔄 Instant message updates without page refresh

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Chatly.git
cd Chatly
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
uvicorn app:app --reload
```

Open your browser and visit:

```text
http://127.0.0.1:8000
```
